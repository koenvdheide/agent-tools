# review-plugins

A Claude Code plugin marketplace for reviewing AI-generated output.

## Install

```text
/plugin marketplace add koenvdheide/review-plugins
```

Then install any plugin from the catalog:

```text
/plugin install claude-reviewer@review-plugins
```

## Plugins

| Plugin | Description |
| --- | --- |
| [`claude-reviewer`](https://github.com/koenvdheide/claude-reviewer) | QA reviewer subagent and `/claude-reviewer:qa` skill for catching miscounts, duplicates, stale totals, hallucinations, and internal contradictions |
